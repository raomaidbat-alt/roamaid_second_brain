# audit_account

## Что делает

Собирает данные аккаунта через instagrapi и social-analyzer, делает аудит по 7 этапам через Gemini, записывает результат в Google Sheets, генерирует 3 варианта персонального сообщения с A/B тестированием.

## Шаги выполнения

1. Извлечь ник из сообщения (убрать @, взять из URL)
2. Получить данные через `POST http://150.241.116.28:8000/profile {"username": "<ник>"}` — bio, посты, подписчики, хэштеги
3. Также запросить `POST http://150.241.116.28:8000/analyze {"url": "https://www.instagram.com/<ник>/"}` — доп. метаданные
4. Отправить данные в Gemini — получить аудит по 7 разделам + 3 варианта сообщений
5. Создать лист аудита в Google Sheets (таблица ID: `1nqBnh8WcEyb9i5B_kt9YmSkQLoBOY-pSnwqE3CJNUuo`)
6. Выбрать вариант сообщения А/Б/В на основе A/B статистики из листа CRM
7. Записать в лист CRM: дата, ник, ссылка на аудит, сообщение, вариант

## 7 разделов аудита

```
1. Продуктовая линейка — оффер, целевая аудитория, линейка продуктов, цены
2. Упаковка профиля IG — шапка, аватар, CTA, понятно ли куда идти
3. Ссылка в профиле — одна/много, есть ли TG бот, лендинг
4. Reels — просмотры, форматы, регулярность, CTA в конце роликов
5. TG канал — есть ли, подписчики, активность, воронка продаж
6. TG бот — автоматизация, приветствие, прогрев, оффер
7. Монетизация — текущий доход, потенциал, что блокирует рост
```

## 3 варианта персонального сообщения

Все сообщения строго на ВЫ. Заполнять реальными данными из аккаунта.

**Вариант А (ЦИФРЫ):**
```
{ник}, доброго дня!
случайно попал на Ваш профиль — {X} подписчиков, {Y} просмотров на рилсах.
Заметил что при таких цифрах Вы могли бы зарабатывать в 2-3 раза больше...
[наблюдение 1], [наблюдение 2], [наблюдение 3]
работал недавно с Абдурап Мурзаев — 10 рилсов, +415к | кейс: [ссылка]
вот небольшой аудит с деталями → {ссылка на таблицу}
посмотрите если интересно)
```

**Вариант Б (БОЛЬ):**
```
{ник}, доброго дня!
случайно попал на Ваш ролик про {тема} — смотрел не отрываясь.
Вижу что контент у Вас сильный, но есть пара вещей которые тормозят монетизацию...
[боль 1], [боль 2], [боль 3]
[кейс + ссылка на таблицу]
```

**Вариант В (КОМПЛИМЕНТ):**
```

случайно попал на Ваш профиль — редко встречаю такой качественный контент в нише {ниша}.
При этом заметил кое-что в воронке что можно докрутить...
[наблюдение 1], [наблюдение 2], [наблюдение 3]
[кейс + ссылка на таблицу]
```

## A/B выбор варианта

- Читать лист CRM Google Sheets, смотреть колонки "Вариант (А/Б/В)" и "Ответил (да/нет)"
- Если один вариант набрал 5+ ответов с конверсией выше других — использовать его в 60% случаев
- Иначе — равномерный случайный выбор

## Google Sheets структура

**Лист "CRM"** — заголовки:
`Дата | Ник | Ссылка на таблицу аудита | Персональное сообщение | Вариант (А/Б/В) | Ответил (да/нет) | Дата ответа | Статус | Оценка | Комментарий`

**Лист аудита** (создаётся отдельно для каждого ника):
`Этап | Наблюдение | Почему просаживает продажи | Как должно быть | Задачи | KPI`
