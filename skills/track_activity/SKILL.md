# track_activity

## Что делает

Принимает голосовое сообщение или текст с перечислением активностей, транскрибирует голос через Whisper, парсит цифры и тип активности, записывает строку в Google Sheets.

## Поддерживаемые активности

| Паттерн | Тип записи |
|---|---|
| `15 звонков`, `сделал 7 звонков` | `звонок` |
| `3 рассылки`, `отправил 50 рассылок` | `рассылка` |
| `2 встречи`, `провёл встречу` | `встреча` |

## Шаги выполнения — голосовое сообщение

1. Скачать аудио файл (`.ogg`)
2. Отправить на `POST http://150.241.116.28:8000/transcribe-file` как multipart/form-data
3. Получить транскрипцию из `response.text`
4. Перейти к шагу парсинга

## Шаги выполнения — текстовое сообщение

1. Взять текст напрямую
2. Применить регулярки для извлечения активностей

## Парсинг активностей

```python
patterns = [
    (r'(\d+)\s*звонк\w*', 'звонок'),
    (r'(\d+)\s*рассылк\w*', 'рассылка'),
    (r'(\d+)\s*встреч\w*', 'встреча'),
]
```

## Запись в Google Sheets

Лист: `Активность`
Заголовки: `дата | тип | количество | заметка`

Записывать одну строку на каждый тип активности. В поле `заметка` — полный текст транскрипции.

## Формат ответа

При успехе: `Записал: 15 звонков, 3 рассылки`
При ошибке парсинга: показать транскрипцию и попросить уточнить формат
При недоступности Whisper: предложить написать текстом
