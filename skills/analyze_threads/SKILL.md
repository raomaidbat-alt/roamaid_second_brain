# analyze_threads

## Что делает

Принимает ссылку на конкретный пост Threads или имя аккаунта (@username).
- Если пост → анализирует именно его
- Если аккаунт → берёт топ-5 постов по лайкам из последних 15

Через Gemini готовит: тип крючка, психологический триггер, структуру, причины виральности.
Генерирует адаптированный тред для ниши "рост в соцсетях".

## Зависимости

- `THREADS_USERNAME`, `THREADS_PASSWORD` в .env
- `GEMINI_API_KEY` в .env
- `threads_api` (1.2.0), `instagrapi`

## Обработка ошибок

- Нет креденшелов → понятное сообщение
- Приватный аккаунт → сообщить пользователю
- 429 от Gemini → 3 попытки с паузой 30 сек
