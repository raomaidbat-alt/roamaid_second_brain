# audit_website

## Что делает

Скачивает HTML страницы, извлекает заголовок/CTA/структуру через BeautifulSoup,
отправляет в Gemini с промптом анализа по 7 разделам, записывает в Google Sheets
лист "Сайты", генерирует 3 варианта персонального сообщения.

## 7 разделов анализа

```
1. Первый экран — оффер, заголовок (H1), CTA кнопки, ясность посыла
2. Воронка — куда ведёт человека, логика шагов, есть ли следующее действие
3. Социальные доказательства — отзывы, кейсы, цифры, логотипы клиентов
4. Продуктовая линейка — что продаётся, цены, офферы, понятность условий
5. Мобильная версия — адаптивность, читаемость на телефоне
6. Скорость и технология — признаки медленной загрузки, iframe, тяжёлые блоки
7. Конверсионность — итоговая оценка 1-10, топ 3 проблемы, потенциал
```

## Шаги выполнения

1. Скачать HTML через requests (User-Agent: Chrome, таймаут 15 сек)
2. Извлечь через BeautifulSoup: title, meta description, h1, h2, кнопки, ссылки, текст body
3. Составить контекст (не более 4000 символов текста)
4. Отправить в Gemini с промптом анализа
5. Распарсить JSON ответ
6. Записать в лист "Сайты" Google Sheets
7. Вернуть структурированный отчёт + 3 варианта сообщений
